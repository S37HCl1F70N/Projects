## 📁 Projects Overview

### 🎨 Dashboard
A customizable, retro-themed daily dashboard for productivity, habit tracking, and quick access to favorite links.

- **Theme Toggle**: Switch between CRT and dark mode themes
- **Quick Links Widget**: Manage and access favorite links
- **Countdown Timer**: Visual timer with progress bar for focused work sessions
- **Modular Widgets**: Extensible widget system for future tools
- **Radar Page**: Integrated NOAA radar dashboard
- **Tech Stack**: HTML, CSS, JavaScript (static - no build required)

**📂 Location**: `/Dashboard/`  
**🎮 Play**: Open `Dashboard/html/dashboard.html` or `Dashboard/html/radar.html` in your browser

## 🚀 Quick Start

### Running the Games
1. Navigate to `/html_games/`
2. Open any `.html` file in your web browser
3. Play!

### Running the Dashboard
1. Open `/Dashboard/html/dashboard.html` in your web browser
2. Use the theme toggle to switch between CRT and dark modes
3. Customize widgets as desired

### Running Penny for Your Time
1. Navigate to `/Penny_for_your_time/`
2. Double-click `Launcher.bat`
3. The UI will open with your time tracking interface

### Running Character Creator
1. Install Python 3 and required packages:
   ```bash
   pip install pandas openpyxl
   ```
2. Generate JSON from the Excel file:
   ```bash
   python3 character_generation.py character_generation_schema.xlsx
   ```
3. Start a local server:
   ```bash
   python3 -m http.server
   ```
4. Open `http://localhost:8000/character_generator.html` in your browser
